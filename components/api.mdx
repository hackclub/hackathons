# API

A simple JSON API to events on this site. If you use it, you must credit
“Hack&nbsp;Club Hackathons” with a link to <https://hackathons.hackclub.com/>.

## Endpoints

- [`/api/events/upcoming`](/api/events/upcoming) – all upcoming events
- [`/api/events/all`](/api/events/all) – all events ever
